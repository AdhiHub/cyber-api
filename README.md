# ☠️ Cyber API

> Encrypt · Decrypt · Hash  > 🔗 **Live Demo:** [https://adhihub.github.io/cyber-api/](https://adhihub.github.io/cyber-api/)

---

## 🔧 What It Does

### 1. 🔐 AES-256 Encryption
Enter text + a secret key → get encrypted output. Use the same key to decrypt.

### 2. 🔢 Hash Generator
Generate SHA-256, SHA-512, or SHA-1 hashes.

### 3. 🔑 Decrypt
Decrypt text back to original using your secret key.

---

## 🚀 How to Use

Open `index.html` in any browser. No installation needed.

Or run a local server:
```bash
npx serve .
```
